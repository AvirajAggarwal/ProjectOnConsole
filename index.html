Visual Studios-Visual Studio is an Integrated Development Environment (IDE) developed by Microsoft to develop GUI (Graphical User Interface), console, Web applications, web apps, mobile apps, cloud, and web services, etc. With the help of this IDE, you can create managed code as well as native code.

Console Screen-Follow the below steps to create the console application using visual studio code: Open Visual Studio Code, then open a project. Then open the Integrated Terminal from Visual Studio Code selecting. Terminal > New Terminal from the sub menu. The dotnet command creates a new application of type console for you.To print on console screen we use the function console.log
